# Douyin Compass Collector Skill

This repository contains a Codex skill for maintaining and installing the Douyin E-commerce Compass live product explain collection workflow.

## Install In Codex

Ask Codex:

```text
Use $skill-installer to install https://github.com/<owner>/<repo>/tree/main/skills/douyin-compass-collector
```

After installation, ask:

```text
Use $douyin-compass-collector to install or maintain the Douyin Compass collector.
```

## Runnable Automation Package

The skill bundles the runnable files under:

`skills/douyin-compass-collector/assets/automation-package`

Copy that folder to a normal working directory, fill `自动化流程配置.xlsx`, then double-click:

`运行抖音罗盘采集.cmd`

## Security

The bundled workbook is a template. Do not commit real account credentials, cookies, session data, or sensitive shop data.
